# Оригинальная документация Paygine API

Положите сюда официальный документ Paygine (PDF или Word) с полным описанием P2PMarket API:

- **Рекомендуемое имя файла:** `Paygine-P2PMarket-API.pdf` или `Paygine-P2PMarket-API.docx`
- После добавления файла интеграцию можно сверять с документом (таблицы параметров, подписи, коды ошибок, приложения).

Файлы в этой папке не коммитят в git, если документ конфиденциальный (добавьте `docs/paygine/*.pdf` и `docs/paygine/*.docx` в `.gitignore`).

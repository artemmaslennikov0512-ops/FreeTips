# Шрифты для PDF-чеков

Для генерации PDF-чеков на русском **обязателен** TTF-шрифт с поддержкой кириллицы. Без него в чеке вместо ФИО и реквизитов будут отображаться знаки вопроса (????).

**Рекомендуемый шрифт:** Noto Sans Regular  
Скачать: https://github.com/google/fonts/raw/main/ofl/notosans/NotoSans-Regular.ttf  
Сохраните файл как: `NotoSans-Regular.ttf` в эту папку (`public/fonts/`).

- **Локально:** если файла нет, приложение попытается загрузить шрифт из интернета при первом запросе чека.
- **Docker/продакшен без интернета:** обязательно положите `NotoSans-Regular.ttf` в `public/fonts/` при сборке образа, иначе чеки с русским текстом формироваться не будут.

**Логотип в PDF:** чтобы в чеке отображался логотип сайта, положите в `public/` файл `logo.png` (PNG, не SVG). Если его нет, в чеке будет выведено название «FreeTips» текстом.
